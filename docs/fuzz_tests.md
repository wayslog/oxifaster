# Fuzz Testing (cargo-fuzz)

This repository uses `cargo-fuzz` (libFuzzer) for fuzzing.

Current target:

- `fasterkv_public_api`: fuzzes `FasterKv` public CRUD/session behavior with a model-checking loop.

## Install

```bash
rustup toolchain install nightly
cargo install cargo-fuzz
```

## Run

Local default (30 minutes):

```bash
./scripts/check-fuzz.sh
```

CI mode (5 minutes):

```bash
./scripts/check-fuzz.sh ci
```

## Tunables

- `OXIFASTER_FUZZ_MAX_TOTAL_TIME` (seconds): total fuzzing duration.
- `OXIFASTER_FUZZ_TIMEOUT` (seconds): timeout for one test input.
- `OXIFASTER_FUZZ_MAX_LEN` (bytes): max input size generated by libFuzzer.
- `OXIFASTER_FUZZ_RSS_LIMIT_MB` (MB): memory limit.

Example:

```bash
OXIFASTER_FUZZ_MAX_TOTAL_TIME=120 OXIFASTER_FUZZ_TIMEOUT=10 ./scripts/check-fuzz.sh
```
