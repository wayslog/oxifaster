# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability in oxifaster, please report it privately.

- Preferred: open a GitHub Security Advisory for this repository
- Include: a detailed description, steps to reproduce, affected versions, and any proof-of-concept

We will acknowledge receipt as soon as possible and work with you to validate and address the issue.

## Supported Versions

Security fixes are applied to the latest released version on the `main` branch.
